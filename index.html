What is DrumScript?

DrumScript is an innovative programming language that combines the art of rhythm with code. Inspired by musical patterns, it allows you to create dynamic drum sequences and control their flow using simple programming syntax. Whether you're a musician, programmer, or enthusiast, DrumScript brings an exciting new way to code and compose rhythms.

Features

  • drum-terminolgy syntax based on rhythm patterns.
  • Generate random drum sequences with customizable rhythms.
  • Integrate various drum parts like kick, snare, hi-hat, cymbal, floor_tom and tom into your code.
  • Perfect for music lovers interested in programming and vice versa!
